Prompt: „Ich habe ein Schulprojekt. Thema: 'Musikschule – Unterrichtsverwaltung'. Bitte erstelle mindestens 10 sinnvolle funktionale Anforderungen für eine Software, mit der Musiklehrer ihre Schüler verwalten, Unterrichtstermine planen und Zahlungen verfolgen können. Formuliere jede Anforderung in einem vollständigen Satz."

1. Das System ermöglicht das Anlegen, Bearbeiten und Löschen von Schülerprofilen mit Angaben wie Name, Kontaktdaten, Instrument und Unterrichtsart.

2. Das System ermöglicht das Anlegen, Bearbeiten und Löschen von Lehrerprofilen mit Angaben wie Name, Kontaktdaten und den unterrichteten Instrumenten.

3. Das System ermöglicht das Planen von Unterrichtsstunden, indem einem Schüler, einem Lehrer, einem Datum, einer Uhrzeit und einer Dauer ein Termin zugeordnet wird.

4. Das System zeigt eine Übersicht aller geplanten Unterrichtsstunden in einer Listen- oder Kalenderansicht gefiltert nach Datum, Lehrer oder Schüler an.

5. Das System ermöglicht das Erfassen des Zahlungsstatus je Unterrichtsstunde (z. B. „offen“, „bezahlt“) für jeden Schüler.

6. Das System ermöglicht das Erstellen eines Monatsüberblicks pro Schüler, in dem alle Stunden und der dazugehörige Zahlungsstatus angezeigt werden.

7. Das System ermöglicht das Markieren von Unterrichtsstunden als „abgesagt“ und speichert optional einen Absagegrund.

8. Das System unterstützt das Suchen von Schülern und Lehrern nach Namen, Instrument oder anderen Kriterien.

9. Das System ermöglicht das Exportieren einer einfachen Liste aller Unterrichtsstunden und Zahlungen eines Monats in eine Text- oder CSV-Datei.

10. Das System erinnert den Lehrer an offene Zahlungen, indem eine Liste aller Schüler mit noch offenen Beträgen angezeigt wird.
